# Hunte

**Hunte** is a turn-based tactical combat game with FPS elements. At the moment, this is just a personal project. 

## Controls

| Action        | Controls                                                       |
|:--------------|:---------------------------------------------------------------|
| Movement      | <kbd>W</kbd><kbd>A</kbd><kbd>S</kbd><kbd>D</kbd> or arrow keys |
| Throw Grenade | <kbd>G</kbd>                                                   |
